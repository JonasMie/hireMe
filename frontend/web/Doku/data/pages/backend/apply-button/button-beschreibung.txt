====== HireMe Button ======

===== Allgemein =====

{{:backend:apply-button:hireme-button-1-dark.png?nolink&200 |}}
Der HireMe Button bietet einerseits für Unternehmen die einfache Generierung sowie Einbettung auf einer Seite, andererseits jedoch auch die Möglichkeit zur "One-Click" Bewerbung für Bewerber. Sobald der Button in eine Seite eingebunden wird, trackt er alle Operationen, die mit ihm ausgeführt werden, sodass die Recruiter der Unternehmen stets alle Informationen innerhalb von Sekunden bereitgestellt bekommen. Der Button wird in folgenden Sektionen beschrieben. Dabei wird unterschieden zwischen 
    * [[backend:apply-button:button-recruiter|HireMe-Button für Recruiter]]
    * [[backend:apply-button:button-bewerber|HireMe-Button für Bewerber]]
    * [[backend:apply-button:button-logic|HireMe-Button Funktionalität und Prozess]]
