====== HireMe-Button für Recruiter ======

Sobald ein Recruiter eine Stellenanzeige erstellt hat, ist es ihm möglich einen dazu gehörigen Button zu erstellen. Dies kann auf der "Stellenanzeige ansehen" Seite erfolgen: 

{{:backend:apply-button:bildschirmfoto_2015-08-18_um_16.15.57.png?nolink|}}

Wie im Screenshot zu sehen ist, besitzt jeder Button einen einmaligen Key, der für das Aufzeichnen von Views, Clicks, sowie der Identifikation der Stellenanzeige dient. Der Recruiter hat nun die Möglichkeit den Code für das Einbinden des Buttons zu kopieren und den Key des Buttons an die Stelle des Platzhalters zu kopieren. In diesem Beispiel würde das nun folgendermaßen aussehen: 

<code html>
<!-- Codezeile zum Laden des Skripts -->
<script src='hireme.mi.hdm-stuttgart.de/js/applier.js'></script>
<!-- Codezeile zum Einbinden des Buttons -->
<div id='ac' name='245de80cc0fae337e92fbe65a09c38a1'></div>
</code>
